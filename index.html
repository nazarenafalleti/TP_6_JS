  7. ¿Qué es el objeto dom en JS? Y ¿Cómo trabaja?.

    8. El dom es como un arbol de objetos, es decir la estructura del documento HTML, que esta formado por muchas etiquetas anidadas dentro de otras y asi se arma el arbol de etiquetas que es llamada DOM.En Javascript, DOM tambien se refiere a una estructura, que podemos modificar desde Javascript, agregando nuevas etiquetas, modificando o eliminando otras, cambiando sus atributos HTML, añadiendo clases, cambiando el contenido de texto, etc.

  8. ¿Qué función tiene en JS el Object Map?.

    9. Un Map,es un grupo de pares clave-valor que se puede usar cualquier tipo de datos como clave y que se encarga de mantener el orden de sus entradas.
